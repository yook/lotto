Об игре

Добро пожаловать в "Музыкальную Рулетку" - интерактивный музыкальный плеер, который случайным образом выбирает треки для вас!

Как это работает?

  • Нажмите кнопку "Крутить рулетку"
  • Система случайно выберет число из доступного диапазона
  • Каждое число соответствует уникальному треку
  • Трек автоматически начнет играть
  • Сыгранные номера сохраняются в списке
  • В разделе «Настройки» можно задать Название игры и установить авто-продолжение — при завершении трека рулетка начнёт крутиться автоматически
  • Кнопка «Новая игра» очищает историю номеров — можно начинать следующий раунд.

Правила игры

  1. Игроки открывают карточку по QR-коду; при входе автоматически создаётся бинго-карта из 75 чисел с отмеченной центральной ячейкой ⭐.
  2. Ведущий нажимает «Крутить рулетку», чтобы получить очередной номер трека.
  3. После каждого выпавшего номера игроки отмечают его на своей карте; номер больше не повторится в текущей игре.
  4. Победа засчитывается за любую заполненную линию: строка, столбец, диагональ или полностью закрытая карта.
